# 🚀 Tokenomics Book React - Startup Guide

## Quick Start Options

### Option 1: Use the Fixed Batch Script (Windows) - Recommended
```bash
# Double-click or run from command line:
start-7500-fixed.bat
```

### Option 2: Use PowerShell Script (Windows)
```powershell
# Right-click and "Run with PowerShell" or run from PowerShell:
.\start-7500.ps1
```

### Option 3: Use PowerShell Script (Windows)
```powershell
# Right-click and "Run with PowerShell" or run from PowerShell:
.\start-7500.ps1
```

### Option 4: Manual Commands
```bash
# Navigate to the project directory
cd tokenomics-book-react

# Install dependencies (if not already installed)
npm install

# Set port and start
set PORT=7500
npm start
```

## Troubleshooting

### If the batch script doesn't work:

1. **Check Node.js Installation**
   ```bash
   node --version
   npm --version
   ```
   If these commands fail, install Node.js from https://nodejs.org/

2. **Check Directory**
   Make sure you're in the `tokenomics-book-react` directory and that `package.json` exists.

3. **Manual Start**
   ```bash
   cd tokenomics-book-react
   npm install
   set PORT=7500
   npm start
   ```

4. **Alternative Port**
   If port 7500 is busy, try:
   ```bash
   set PORT=3001
   npm start
   ```

### Common Issues:

- **"npm is not recognized"**: Install Node.js
- **"Port already in use"**: Try a different port or close other applications
- **"Permission denied"**: Run as administrator or check file permissions
- **"Module not found"**: Run `npm install` to install dependencies

## Accessing the Application

Once started successfully, open your browser and go to:
- **Main URL**: http://localhost:7500
- **Glossary**: http://localhost:7500/glossary
- **Tooltip Example**: http://localhost:7500/glossary-example

## Features Available

- 📚 **Complete Glossary** with 80+ terms
- 💡 **Interactive Tooltips** on hover
- 🎮 **Simulations** and **Quizzes**
- 🌙 **Dark/Light Theme** toggle
- ⚙️ **Settings Panel** for customization

## Stopping the Server

Press `Ctrl+C` in the terminal/command prompt to stop the development server.
